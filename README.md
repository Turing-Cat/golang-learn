# Golang Learn

Go 语言学习项目，包含系统化学习路线、代码练习和项目实战。

## 学习阶段

- **阶段 1**：Go 基础语法与程序结构
- **阶段 2**：Go 核心抽象与语言特性
- **阶段 3**：工程能力与标准库
- **阶段 4**：项目实战

## 学习目标

基于已有编程基础（Python），系统掌握 Go 语言，达到独立开发中小型后端项目的能力。
