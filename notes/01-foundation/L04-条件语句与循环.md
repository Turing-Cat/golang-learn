# L04 条件语句与循环

## 1. 本节目标

掌握 Go 的三大控制流：if/else、switch、for。

## 2. 核心概念

- **if/else**：条件不加括号，`{` 必须同行；支持 `if 初始化语句; 条件 {}` 缩小变量作用域
- **switch**：自动 break（不用手写）；case 可多值（逗号分隔）；可无参替代 if-else 链
- **for**：Go 只有 for，没有 while
  - 经典三段式：`for i := 0; i < n; i++ {}`
  - 当 while 用：`for 条件 {}`
  - 死循环：`for {}`，用 break 跳出
- **break** 跳出循环，**continue** 跳过本轮

## 3. 最小示例

```go
// if 带初始化语句
if score := 85; score >= 60 {
    fmt.Println("及格")
}

// switch 无参形式
switch {
case score >= 90:
    fmt.Println("优秀")
case score >= 60:
    fmt.Println("及格")
default:
    fmt.Println("不及格")
}

// for 当 while
n := 1
for n < 100 {
    n *= 2
}
```

## 4. 易错点

- `{` 换行是**编译错误**，不是风格问题
- switch case 从上到下匹配，顺序影响结果
- for 条件循环可能一次都不执行（如初始条件就为 false）

## 5. 一句话总结

Go 的控制流精简但完备：if 不加括号，switch 自动 break，for 包打天下。

## 6. 复习时间

学到函数后，回来用 for + if 组合写更复杂的逻辑时复习。
