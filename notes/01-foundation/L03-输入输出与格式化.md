# L03 输入输出与格式化打印

## 1. 本节目标

掌握 Go 的 fmt 包三大功能：输出、格式化、输入。

## 2. 核心概念

- **Print 系列**：`Print`（不换行）、`Println`（自动换行）、`Printf`（格式化，不自动换行！）
- **Sprintf**：格式化但不打印，返回字符串
- **Scan 系列**：`Scanln(&var)` 读取用户输入，`&` 是取地址符
- **常用格式动词**：`%v`（默认）、`%d`（整数）、`%s`（字符串）、`%f`（浮点数）、`%.2f`（保留2位小数）、`%T`（类型）、`%%`（百分号本身）
- **Go 没有隐式类型转换**：`float64 * int` 不行，必须手动 `float64(count)`

## 3. 最小示例

```go
name := "Go"
version := 1.22
fmt.Printf("%s 版本 %.2f\n", name, version)
```

## 4. 易错点

- `Printf` 不会自动加换行，必须手动加 `\n`
- 格式动词必须和变量类型匹配，`%d` 配 float64 会输出 `%!d(float64=...)`
- 跨类型运算需要显式转换：`float64(count)`

## 5. 一句话总结

fmt 包是 Go 最常用的 I/O 工具，记住 Printf 要手动加 \n，类型转换不能偷懒。
